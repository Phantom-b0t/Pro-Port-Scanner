 Python Pro Port Scanner

A high-performance, multi-threaded port scanner built with Python. 

## Features
- **Multi-threading:** Scans 50+ ports simultaneously for maximum speed.
- **Service Detection:** Identifies services using standard socket libraries.
- **Banner Grabbing:** Extracts server information to guess the target OS.
- **Output Logging:** Automatically saves results to `results.txt`.

## How to Run
1. Clone the repo: `git clone https://github.com/AapkaUsername/Pro-Port-Scanner.git`
2. Run the script: `python port_scanner.py`
3. Enter the target IP and port range.
